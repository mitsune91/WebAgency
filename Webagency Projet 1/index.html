UX design
C'est l'expérience de l'utilisateur (User eXperience en anglais). C'est une discipline qui prend en compte les attentes et les besoins de l'utilisateur pour créer un site web ou une application.
UI design
C'est l'interface utilisateur (User Interface en anglais). C'est ce qui permet d'ameliorer l'interaction d'un utilisateur avec le produit.
SEO
Search Engine Optimization en anglais peut etre defini comme l'art de positionner un site, une page web ou une application dans les moteurs de recherche.
